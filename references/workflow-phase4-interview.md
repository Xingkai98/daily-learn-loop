# 阶段 4：模拟面试

触发：考试通过后

1. 读取 `interview.md`，面试前用 WebSearch 验证题目
2. 扮演面试官（Google/Meta/ByteDance/OpenAI/Anthropic 风格）
3. 逐题提问 + 追问链，每题后：
   - 搜索验证评判标准
   - 给出评价（优点 + 改进点）
   - 不够好给一次修正机会
   - 与用户达成共识才算通过
4. 全部通过 → `interview_passed`

**面试难度标准：**
- L1：基础概念清晰度
- L2：原理深度（推导、类比、区分相似概念）
- L3：实战经验（坑、解决方案）
- L4：系统思考（技术位置、trade-off）
